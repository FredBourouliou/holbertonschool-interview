# Star Wars API

Script that prints all characters of a Star Wars movie using the Star Wars API.

## Usage

```bash
./0-starwars_characters.js <movie_id>
```

## Example

```bash
./0-starwars_characters.js 3
```

Prints all characters from "Return of the Jedi" in order.

## Requirements

- Node.js 10.x
- request module

## Installation

```bash
npm install request
```
